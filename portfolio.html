Load Frequency Control

Designed a Load Frequency Control for a single synchronous generator implementing digital PID control in MATLAB and Simulink.

  • Found the Transfer Functions of Governor, Turbine and Generator with given data provided using System Identification tool.
  • Designed and tuned a continuous PID and developed a discrete PID controller accordingly with different transformation methods with the optimal sampling time.
